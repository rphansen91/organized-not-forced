# Landing Page & Email Setup Guide

## Overview

You need two things:
1. **Landing page** — A simple page to capture emails
2. **Email service** — To manage subscribers and send updates

Both can be set up for **free** in under an hour.

---

## Option 1: Carrd + Beehiiv (Recommended)

### Why This Combo
- **Carrd** — Beautiful one-page sites, free tier works great
- **Beehiiv** — Modern email platform, great free tier, built for creators

### Step 1: Set Up Beehiiv (Email)

1. Go to [beehiiv.com](https://beehiiv.com)
2. Click "Start Writing for Free"
3. Create account
4. Name your publication: "Organized, Not Forced" or "[Your Name] — Calisthenics Journey"
5. Get your form embed code (Settings → Subscribe Forms)

### Step 2: Set Up Carrd (Landing Page)

1. Go to [carrd.co](https://carrd.co)
2. Click "Start" → Sign up free
3. Choose "Pro Lite" template style (or start blank)
4. Build your page (see copy below)
5. Embed Beehiiv form
6. Publish with free carrd.co subdomain (e.g., organizednotforced.carrd.co)

### Ongoing: Weekly Newsletter

**Send every Sunday or Monday**

**Subject ideas:**
- "Week [X]: What I learned"
- "[Skill] breakthrough — here's what happened"
- "The principle I can't stop thinking about"
- "Chapter [X] preview"

**Structure:**
1. Training update (what you worked on)
2. One insight or principle
3. One struggle or question
4. Book progress update
5. Call to action
